Linux Exercise

  1. Download the file to your computer from the following url Mesculenta_147_gene.gff3.gz

  2. Log into greene.hpc.nyu.edu.

  3. Create a directory on greene.hpc.nyu.edu called LinuxExercise

  4. Transfer the file Mesculenta_147_gene.gff3.gz to your new directory

  5. Use gunzip to uncompress the file.

  6. Rename the file to : Cassavagenes.txt

  7. Use the head command to look at the first 20 lines of Cassavagenes.txt

  8. Determine the number of different types of features that are present in the file.

    1. Hint – use the cut function to select the column with features.

    2. Hint - You can use the uniq function but first you must sort it.

  9. Save the rows that contain the word “gene” as a separate file. Call the new file “Genes.txt”

    1. Hint - use the grep command to search for pattern “gene”

  10. How many genes are in Genes.txt ? 

    1. Hint – count the number of lines using wc.
